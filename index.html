Welcome to AHAR

AHAR(Autonomous Home Assitant Robot) has been designed in a way that it can assist in household chores. We have designed it in such a manner that it is highly customizable. The user can have it configured according to their requirements. We built this project as a part of our Final Year Mechanical Engineering Project.


Project Motivation

Our primary intent was to build a project that can help the society. An assistant robot can be used in multiple applications such as to monitor a patientâ€™s health, as an assistant in workshops, etc. Our robot is capable of helping out with household chores. Our current design performs the task of serving a glass of water (liquids, in general). As stated earlier, this design can be used to perform different tasks as well. A home assistant robot will prove to be highly beneficial to senior citizens, working people, people with disability, etc. We believe that in the future the use of home assistant robots specifically will become as common as using a smartphone today.
